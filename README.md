# Aarogya AI
India's first two-sided health intelligence platform built for low-income communities.

## What it does
- Predicts disease from symptoms using a 99.3% accurate Naive Bayes model
- Recommends nearby doctors sorted by distance, price and rating
- Detects community disease outbreaks for ASHA workers and NGOs

## Tech Stack
- RapidMiner AI Studio — predictive modeling
- Mendix Studio Pro — application interface

## Dataset
- Disease Symptom Prediction (Pranay Patil, Kaggle) — 4,920 examples, 41 diseases
- Doctor Specialty Recommendation (Ebrahim Elgazar, Kaggle)
